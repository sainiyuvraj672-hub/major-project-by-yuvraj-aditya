# 🗳️ FairVote: Smart Voting System

**FairVote** is a secure, scalable, and transparent digital voting platform built using modern web technologies to ensure fair and efficient elections. Developed as a **BCA Major Project at Manipal University Jaipur**, this system aims to eliminate electoral fraud and streamline the voting process.

---

## 🚀 Overview
FairVote is a full-stack web application designed to replace traditional, paper-based, or insecure voting methods with a robust digital solution. 

### Core Pillars:
*   ✅ One Person, One Vote
*   🔐 Biometric Authentication
*   ⚡ Real-Time Results
*   📊 Transparency

---

## 🧠 Key Features
*   Biometric Authentication using WebRTC
*   Digital Voter ID generation using jsPDF
*   Secure Voting Booth with duplicate prevention
*   Admin Dashboard for election control
*   Multi-Tenant election support
